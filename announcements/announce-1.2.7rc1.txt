Hello, ProFTPD community. The ProFTPD Project team is pleased to announce
that the first release candidate for ProFTPD 1.2.7 is now available
for public consumption.

You can download 1.2.7rc1, including PGP signatures and MD5 sums, from any
of the proftpd mirrors.  Mirrors are available via FTP as

  ftp.<two_letter_iso_country_code>.proftpd.org

(example: ftp.nl.proftpd.org).  Not all countries have mirrors; however
you should select one that is geographically close to you.

Alternatively, you can download proftpd from the main site:

  ftp://ftp.proftpd.org/distrib/source

RPMs, once available, will be placed here:

  ftp://ftp.proftpd.org/distrib/packages/RPMS

The 1.2.7rc1 release includes numerous new features, including:

  - New configuration directives:
       AllowOverride:
         Toggle parsing of .ftpaccess files.

       DefaultAddress:
         Configure address of default server; for multi-homed machines.

       HideFiles:
         Hide files based on regex matching of file names.

       MaxClientsPerUser:
         Limit number of simultaneous connections per user.

       MaxConnectionRate:
         Connection rate throttling; for standlone daemons.

       MaxRetrieveFileSize, MaxStoreFileSize:
         Limit the maximum size of individual files being transferred

  - Support for a SITE CHGRP FTP command

  - New mod_sql, version 4.10:
     This version supports a new SQLNegativeCache directive

  - A new authentication contrib module: mod_radius

  - Optimizations when listing directories, transferring ASCII files.

Please read the included NEWS and ChangeLog files for the full details.
